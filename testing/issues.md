 Анализ обратной связи пользователей

 1. Метод
Была создана Google Form с 7 вопросами:
- понятность интерфейса
- удобство управления
- стабильность работы
- скорость ответа
- понятность результата
- желание использовать в будущем
- предложения по улучшению

Всего респондентов: 8 человек.

2. Основные результаты

| Показатель | Средний балл (из 5) |
|------------|-----------------------|
| Удобство интерфейса | 4.1 |
| Понятность логики | 4.3 |
| Стабильность | 4.0 |
| Скорость | 4.5 |
| Сложность использования | 1.6 (где 1 — легко) |

 3. Основные комментарии
- «Хотелось бы видеть цветовую шкалу приоритетов»
- «Можно сделать более крупные кнопки»
- «Добавить пояснение формулы»
- «Хочу экспорт результатов»

## 4. Выводы
Пользователи в целом довольны (средний балл 4.2/5).  
Рекомендуется улучшить интерфейс и добавить визуализацию результата.
