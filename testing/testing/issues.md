Список найденных ошибок и предложений

 Ошибки
1. Некоторое время появляется задержка при отправке нескольких сообщений подряд.
2. При вводе текста вместо числа вывод сообщения немного сливается с интерфейсом.
3. На маленьких экранах кнопка “Назад” перекрывает текст.

 Предложения по улучшению
1. Добавить цветовую шкалу (красный-желтый-зелёный).
2. Добавить подсказки к каждому параметру.
3. Улучшить адаптацию под мобильные устройства.
4. Сделать отдельный экран с формулой расчёта.
5. Добавить возможность экспорта в PDF.
